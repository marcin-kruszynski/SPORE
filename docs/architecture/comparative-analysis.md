# Comparative Analysis

## Overstory

Strongest contribution: hierarchical delegation, isolated execution contexts, mail-style coordination, and aggressive observability.

## Gastown

Strongest contribution: durable operator workflow, persistent agent identity, tmux-first live sessions, and explicit workspace topology.

## Mulch

Strongest contribution: structured, explicit knowledge capture as a separate passive layer.

## Beads

Strongest contribution: dependency-aware work graphs and durable task state with event history.

## PI

Strongest contribution: embeddable runtime foundation spanning model access, stateful agents, terminal interaction, and web UI.

## Agentic Engineering Book

Strongest contribution: disciplined prompt, context, and tool-use thinking that supports incremental, inspectable agent systems.

## SPORE Synthesis

SPORE should combine:

- Overstory's bounded hierarchy and coordination rigor,
- Gastown's session and operator model,
- Mulch's explicit knowledge layer,
- Beads-like durable planning concepts,
- PI's runtime and client primitives,
- disciplined documentation and engineering habits from the book.
