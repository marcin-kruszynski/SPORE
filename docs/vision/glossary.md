# Glossary

- Orchestrator: top-level coordination role.
- Lead: domain/substream coordinator.
- Scout: research and mapping role.
- Builder: implementation role.
- Tester: verification role.
- Reviewer: independent quality gate.
- Profile: role behavior package with tools, skills, policies.
- Workflow template: declarative execution path definition.
- Domain: a bounded area of responsibility.
- Session: live execution context of an agent instance.
- Knowledge layer: documentation, decisions, and research corpus.
