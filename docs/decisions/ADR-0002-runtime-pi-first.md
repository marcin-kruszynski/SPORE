# ADR-0002: PI-First Runtime Foundation

- Status: Accepted
- Date: 2026-03-08

## Context

SPORE needs a practical runtime foundation with extensibility and future UI/session integration.

## Decision

Adopt PI ecosystem as primary runtime foundation for early implementation waves.

## Consequences

- Reduced integration uncertainty.
- Strong package-level leverage.
- Future runtime abstraction remains possible but is deferred.
