# ADR-0001: Repository Foundation and Documentation OS

- Status: Accepted
- Date: 2026-03-08

## Context

Initial repository state lacked structural and governance scaffolding.

## Decision

Establish docs-first operating system with explicit folders for architecture, research, decisions, templates, operations, config, workspace, schemas, packages, tools, and references.

## Consequences

- Clear contribution map for agents and humans.
- Higher initial setup cost but lower long-term coordination entropy.
