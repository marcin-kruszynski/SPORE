# Reference Synthesis

SPORE uses the reference repositories as conceptual inputs rather than source material to merge directly.

## Overstory

Borrow the hierarchy, runtime adapter thinking, message-driven coordination, and observability discipline.

## Gastown

Borrow the durable workspace metaphor, persistent identities with ephemeral sessions, and operator-first monitoring model.

## Mulch

Borrow the idea that expertise capture is explicit, local, structured, and separate from runtime execution.

## Beads

Borrow dependency-aware planning, durable work state, and evented task lifecycle concepts.

## PI

Borrow the runtime substrate and client building blocks, especially agent state, session handling, TUI, and web surfaces.

## Agentic Engineering Book

Borrow the emphasis on context control, prompt discipline, and documented operating practice.
