# References

This folder contains local read-only mirrors of upstream repositories used as study material during SPORE design.

## Rules

- Use these repositories for analysis and concept extraction.
- Do not copy code blindly into SPORE.
- Update `REFERENCE_MANIFEST.md` when mirrors are refreshed.
- Capture important lessons in `docs/research/` and `docs/references/reference-synthesis.md`.
